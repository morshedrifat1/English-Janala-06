Frequently Asked Questions

The difference between var, let, and const?

var: (Old Way to Declare Variables)

  • ✔️ Used before ES6.
  • ✔️ Function-scoped (only works inside functions)
  • ✔️ Can be changed and redeclared

let (Modern way to Declare Variables)

  • ✔️ Introduced in ES6.
  • ✔️ Block-scoped (works only inside {} blocks).
  • ✔️ Can be changed

const: (For fixed values)

  • ✔️ Block-scoped, like let
  • ✔️ Hoisted but not initialized.
  • ✔️ Cannot be reassigned (immutable).

The difference between map(), forEach(), and filter()?

map() – Creates a New Array

  • ✔️ Returns a new array with modified values.
  • ✔️ Does not change the original array.
  • ✔️ Best for transforming data.

forEach() – Just Loops, No Return

  • ✔️ Does NOT return a new array.
  • ✔️ Changes the original array if modified inside.
  • ✔️ Best for performing actions (e.g., logging, updating DOM).

filter() – Returns a New Array with Some Items

  • ✔️ Returns a new array with elements that pass a condition.
  • ✔️ Best for filtering data.
Explain arrow functions and how they are different from regular functions?

Arrow functions (=>) are a shorter way to write functions in JavaScript. Arrow functions remove function keyword and use => instead.

example :
----- normal function ----
function sum(num1,num2){
const result = num1 + num2;
}

----- arrow function ----
const sum = (num1,num2) => {
const result = num1 + num2;
}

How JavaScript Promises work?

A Promise in JavaScript is used to handle asynchronous operations. It helps to execute code that takes time (like fetching data from an API) without stopping the rest of the program.

What is a Promise?

A Promise is an object that represents the eventual completion or failure of an asynchronous operation.

It has 3 states:

  • ✔️ Pending → Initial state (not yet completed).
  • ✔️ Fulfilled → The operation was successful.
  • ✔️ Rejected → The operation failed.

How closures work in JavaScript?

A closure is when a function remembers the variables from its outer scope, even after the outer function has finished running.

How Closures Work?

A closure is created when:

  • ✔️ A function is inside another function.
  • ✔️ The inner function uses variables from the outer function.
  • ✔️ The inner function is returned or stored, keeping access to the outer variables.
